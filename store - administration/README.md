# Store Administration - Small Shop
A realistic small shop store administration web application built with *HTML, CSS, and JavaScript*.  
Designed for managing products, inventory, and basic shop operations in a simple, user-friendly interface.

## Features
- Add, Edit, and Delete products
- Upload product images
- Grid view layout like a real small shop
- Data persists in the browser using localStorage
- Lightweight and easy to use

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla)
- LocalStorage for data persistence

## How to Use
1. Go to the store-admin-small-shop folder.  
2. Open index.html in any modern web browser.  
3. Start adding products, updating them, or deleting as needed.  
4. All data is saved in your browser, no backend or database required.
