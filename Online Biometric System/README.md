# Online Biometric System

## Description
This is a *front-end simulation of an online biometric system* designed for small stores or offices.  
It allows users to *login, simulate a biometric fingerprint scan, and record attendance* in a dynamic table.  

This project is built using *HTML, Bootstrap, and JavaScript*, and runs entirely in the browser — no backend required.  

---

## Features
- *User Login:* Predefined users can log in to the system.  
- *Biometric Scan:* Simulate fingerprint scanning with a visual icon.  
- *Attendance Table:* Records attendance for each scan dynamically.  
- *Responsive Design:* Bootstrap ensures the web app looks professional on any device.  
- *Portfolio Ready:* Single-page application suitable for GitHub showcase.

## How to Run
1. Download or clone this repository.  
2. Open index.html in your preferred browser.  
3. Login using one of the predefined users.  
4. Click the fingerprint icon to simulate a biometric scan.  
5. Observe attendance being added to the table.
